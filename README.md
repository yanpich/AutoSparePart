# 🚗 AutoSparePart REST API

## 📌 Project Overview
AutoSparePart is a backend RESTful API developed using Java Spring Boot for managing vehicle spare parts.  
The system provides secure, scalable, and maintainable APIs with JWT-based authentication.

---

## 🚀 Technologies
- Java (Spring Boot)
- Spring Security
- JWT Authentication
- MySQL
- JPA / Hibernate
- Maven

---

## ✨ Features
- User Authentication (Register, JWT Login)
- CRUD Operations (Vehicle, Category, Product, etc.)
- Flexible Search & Filtering (e.g. by model, year range)
- Secure API with validation and error handling
